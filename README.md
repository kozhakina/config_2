# Инструмент визуализации графа зависимостей (Maven-пакетов)

Реализация CLI-инструмента для анализа и визуализации графа зависимостей Java-пакетов (в формате Maven), **без использования сторонних менеджеров пакетов или библиотек получения зависимостей**.

---

## Общее описание

Проект состоит из 5 поэтапно реализованных модулей:
1. **Конфигурация через CLI** — настройка параметров запуска.
2. **Сбор данных** — извлечение прямых зависимостей из `pom.xml` (remote) или JSON-мока (local).
3. **Построение прямого графа** — BFS с обработкой транзитивных зависимостей и циклов.
4. **Обратные зависимости** — поиск пакетов, зависящих от заданного.
5. **Визуализация** — генерация PlantUML-диаграмм, PNG и ASCII-дерева.

Все этапы реализованы «с нуля»: разбор XML/JSON, обход графа, обнаружение циклов, фильтрация — только собственный код.
